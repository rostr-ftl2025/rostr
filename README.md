# Rostr.

## Introduction
Ever wondered how you could pick the ultimate fantasy baseball draft? Or even spent hours deciding on the best players for your team? But can’t find the time to learn all the intricacies of baseball? Well, now you don’t have to--and picking your draft has never been so easy, with Rostr. Our extensive database of pitchers provides you with all the information you need to make the right choices. We recommend the pitchers, and all you have to do is choose. For the baseball fanatics who like to dig deep into each player, we provide the statistics too--all in one place! Anyone can use Rostr, amateur and professional fantasy baseball managers alike.

## Hosting/installation instructions

This project uses [Docker](https://www.docker.com/) to manage containers. Install from: [https://www.docker.com/](https://www.docker.com/).

From the **project directory**, run:

```bash
docker compose up --build
```

Other common commands:

To stop containers:

```bash
docker compose down
```

Clean up Docker environment:
```bash
docker system prune -a
```
